Class ActiveRecord::PreparedStatementInvalid < ActiveRecord::ActiveRecordError

  • /usr/local/lib64/ruby/gems/2.1.0/gems/activerecord-4.2.3/lib/active_record/errors.rb

Raised when number of bind variables in statement given to :condition key (for example, when using find method) does not match number of expected values supplied.

For example, when there are two placeholders with only one value supplied:

Location.where("lat = ? AND lng = ?", 53.7362)
